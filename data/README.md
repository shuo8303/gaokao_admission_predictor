# 官方数据目录

请将浙江省教育考试院公布的普通类一段平行投档数据文件放在本目录。

当前数据读取模块支持：

- .xlsx
- .xls
- .csv

考试院公布的投档线表格建议使用以下列顺序：

1. 学校代号
2. 学校名称
3. 专业代号
4. 专业名称
5. 计划数
6. 分数线，即最低分数
7. 位次，即最低位次

后续精准预测的志愿录入表格将按以下列顺序读取：

1. 学校代号
2. 学校名称
3. 专业代号
4. 专业名称

读取 .xls 文件需要安装 xlrd，部署时请在 requirements.txt 中加入该依赖。

分数位次校验表请放入：

```text
data/score_rank/
```

列顺序：

1. 分数
2. 最高位次
3. 最低位次

系统会用该表校验用户输入的成绩与位次是否匹配。更换年份时，只需要替换
`data/score_rank/` 目录中的表格文件。
